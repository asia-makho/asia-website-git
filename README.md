# 🌍 Weather & Time Dashboard Portfolio

Hi there! This is my interactive portfolio designed to showcase modern web development skills. This project has evolved from a simple static card into a fully dynamic web application.

## 🚀 Key Features
* **Dynamic Weather:** Integrated with the OpenWeatherMap API to fetch real-time data, including temperature, humidity, and wind speed.
* **Analog Clock:** Built using vanilla JavaScript and powered by CSS transformations for smooth movement.
* **Modern UI:** Designed in a Soft Minimalism style, featuring fluid animations and a fully responsive layout.
* **Interactive Contacts:** Animated social media icons and user-friendly navigation.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox, Keyframe Animations).
* **JavaScript (ES6+):** *Asynchronous programming (**Fetch API**, **async/await**).
  * DOM Manipulation.
  * Event Listeners.
* **API:** OpenWeatherMap API.

## 📸 Project Structure
The project consists of three main sections:
Home: Information regarding my education, background, and technical skills.
Contacts: Quick and easy links to get in touch with me.
Weather & Time: A functional dashboard featuring the live widgets.
